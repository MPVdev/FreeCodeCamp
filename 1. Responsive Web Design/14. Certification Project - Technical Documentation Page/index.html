Instalacion

Windows

  1. Instalar un compilador: Descarga e instala MinGW
  2. Agrega la ruta del compilador: (generalmente C:\MinGW\bin) a la variable de entorno PATH.
  3. Instalar un IDE: Descarga e instala Code::Blocks.
Sintasix

La sintaxis de C++ define las reglas de cómo escribir el código para que el compilador lo entienda.

Estructura Basica

#include <iostream>

int main() {
    std::cout << "Hola, Mundo!" << std::endl;
    return 0;
}
  • #include <iostream>: Incluye la biblioteca estándar de entrada/salida.
  • int main() { ... }: Define la función principal del programa.
  • std::cout: Utilizado para imprimir en la consola.
  • std::endl: Inserta un salto de línea.

Variables

Las variables son contenedores para almacenar datos que pueden cambiar durante la ejecución del programa.

Tipos de Datos

  • Enteros: int
  • Decimales: float, double
  • Caracteres: char
  • Booleanos: bool
  • Cadenas: std::string (requiere #include <string>)

Declaración y Asignación

int entero = 10;
float decimal = 5.5;
char caracter = 'A';
bool booleano = true;
std::string cadena = "Hola";
Sentencias Logicas

Las sentencias lógicas permiten tomar decisiones en el flujo del programa basándose en condiciones. se agrupan en 2 tipos de sentencias.

Condicionales

Las condicionales permiten realizar comprobaciones y toma de decisiones.

int x = 10;
if (x > 0) {
  std::cout << "Positivo" << std::endl;
} else if (x < 0) {
  std::cout << "Negativo" << std::endl;
} else {
  std::cout << "Cero" << std::endl;
}

En este ejemplo se comprobaria si la variable x que tiene el valor 10 es mayor que 0 se imprima "Positivo" si no es asi compruebe si es menor que 0 y se imprima "Negativo" y por ultimo si no se cumple ninguna de las condiciones se imprima "Cero".

Bucles

Los bucles son sentencias que se repetiran hasta que la condicion se cumpla.

for (int i = 0; i < 10; i++) {
  std::cout << i << std::endl;
}

El bucle for se inicializa con una variable i en 0 se realiza la sentencia que en este caso es que sea menor a 10 y por ultimo el cambio de variable. esto codigo imprimira la variable i mientras la misma sea menor de 10.

int i = 0;
while (i < 10) {
  std::cout << i << std::endl;
  i++;
}

El bucle while solo comprobara una sentencia y mientras sea verdadera repetira el codigo q tenga dentro, el codigo hace lo mismo que el codigo anterior con la diferencia que si no se agrega el cambio de variable el codigo se repetira infinitamente.

int i = 0;
do {
  std::cout << i << std::endl;
  i++;
} while (i < 10);

El bucle do-while funciona de forma parecida al bucle while con la unica diferencia que primero ejecutara el codigo y despues se comprobara la sentencia esto te da un beneficio en casos que necesites que se ejecute el codigo por lo menos una vez.

Funciones

Las funciones permiten agrupar código reutilizable y organizar mejor el programa.

int suma(int a, int b) {
  return a + b;
  }

Para definir una funcion se debe definir con un tipo de variable que es el tipo de variable que devolvera seguido de un nombre que la defina, adicional se puede definir datos de entrada definidos por un tipo de y seguido por un nombre. dentro de la funcion se realizaran las operaciones y terminaran con un return que devolvera el dato final.

int resultado = suma(6, 3);

std::cout << "El resultado es " << resultado << std::endl;
//esto devolvera "El resultado es 9"
Programacion Orientada a Objetos

La programación orientada a objetos es un modelo de programación en el que el diseño de software se organiza alrededor de datos u objetos, en vez de usar funciones y lógica. Se enfoca en los objetos que los programadores necesitan manipular, en lugar de centrarse en la lógica necesaria para esa manipulación.

class Persona {
private:
  std::string nombre;
  int edad;
 public:
  void establecerNombre(std::string nombre) {
      this->nombre = nombre;
  }
  std::string obtenerNombre() {
    return nombre;
  }
};

Las clases son plantillas para crear objetos, que son instancias de clases.

class Animal {
public:
  virtual void hacerSonido() {
    std::cout << "Sonido de animal..." << std::endl;
  }
};

 class Perro : public Animal {
 public:
  void hacerSonido() override {
    std::cout << "Guau guau!" << std::endl;
  }
};

La herencia permite crear nuevas clases basadas en clases existentes. El polimorfismo permite tratar objetos de diferentes clases derivadas como objetos de una clase base común.

Librerias

Las librerías en C++ permiten usar código preescrito para realizar tareas comunes sin tener que escribir todo desde cero.

#include <vector>
#include <map>

std::vector<int> numeros = {1, 2, 3, 4};
std::map<int, std::string> mapa = {{1, "uno"}, {2, "dos"}};

En este ejemplo se muestra 2 librerias insertadas siguiendo la sintaxis que es la palabra reservada #include seguida de la libreria.
